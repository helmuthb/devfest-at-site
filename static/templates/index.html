{% trans %} DevFests are large-scale community events throughout the world, following the tradition of Google DevFests and organized by local user groups. {% endtrans %}

>>> {% trans %}Videos & Slides from Day 1 are Online!{% endtrans %} <<<

{% trans %} In 2012 there was for the first time a DevFest in Vienna, jointly organized by the Java Student User Group and the Google Developer Group Vienna. {% endtrans %} {% trans %} A total of 145 participants enjoyed a weekend around development for mobile and web. {% endtrans %}

{% trans %}Day 1: Mobile & Web, Cloud & Social{% endtrans %}

{% trans %} The first day was full of workshops and sessions given by members of the local developer community. {% endtrans %} {% trans %} In two rooms at the University of Technology in Vienna sessions were held in parallel. Some of the sessions include: {% endtrans %}
{% trans %} Google+ API family, mobile payment APIs, Go-start web framework, Bitcoin Java API, Google App Scripts, ... {% endtrans %}
{% trans %} Additionally there was enough time to meet and exchange with fellow developers; and when developers come together amazing things can happen. {% endtrans %}

{% trans %} Day 1 endet with an awesome afterparty powered by Cenarion. {% endtrans %}

{% trans %}Day 2: Happy Birthday, Android!{% endtrans %}

{% trans %} It was on 5 November 2007 that Google launched Android as part of the Open Handset Alliance. Now, five years later, we were celebrating its success with a special birthday hackathon! {% endtrans %}
{% trans %} In teams we were developing applications for Android and the winners took home valuable prizes: Nexus 7 tablets, Raspberry Pi's, Arduino Kits, books and gadgets. The hackathon took place in the Metalab, a hacker space in Vienna. {% endtrans %}
