{% trans %} DevFests are large-scale community events throughout the world, following the tradition of Google DevFests and organized by local user groups. {% endtrans %}

{% trans %} This year there is for the first time a DevFest in Vienna, jointly organized by the Java Student User Group and the Google Developer Group Vienna. {% endtrans %}

{% trans %} DevFest Vienna takes place on two days, 10. + 11. November 2012. {% endtrans %}

{% trans %}Day 1: Mobile & Web, Cloud & Social{% endtrans %}

{% trans %} The first day is full of workshops and sessions given by members of the local developer community. {% endtrans %} {% trans %} In two rooms at the University of Technology in Vienna sessions will be held in parallel. Some of the sessions which have been announced are: {% endtrans %}
{% trans %} Google+ API family, mobile payment APIs, Go-start web framework, Bitcoin Java API, Google App Scripts, ... {% endtrans %}
{% trans %} Additionally there is enough time to meet and exchange with fellow developers; and when developers come together amazing things can happen. {% endtrans %}

{% trans %}Day 2: Happy Birthday, Android!{% endtrans %}

{% trans %} It was on 5 November 2007 that Google launched Android as part of the Open Handset Alliance. Now, five years later, we are celebrating its success with a special birthday hackathon! {% endtrans %}
{% trans %} In teams we are developing applications for Android and the winners will take home valuable prizes. The hackathon takes place in the Metalab, a hacker space in Vienna. {% endtrans %}
